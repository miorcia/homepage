Deklarowanie zmiennych

const - stala wartosc

let - zmienna ktora moze zmieniac swoja wartosc, nie naduzywamy let, np. isHappy=true;, isHappy=false;, isHappy='Tak';

JavaScript jest językiem nietypowanym, co oznacza, że zmienna łyknie wszystko

typeof zmienna - zwraca informacje jakiego typu jest zmienna

immutability - niemutowalnosc - fundamentalny koncept JS - nie mozemy zmieniac wartosci primitive values (np. liczby), to co zmieniamy to polaczenie miedzy nimi a zmiennymi

DOM

w JS: document.querySelector('.selector') jest do wysweitlania selektora np. headingu albo elementu opisanego class, zwraca node (wezel) z HTMLa

w jQuery: $('.selector')

w BEM: block__element--js - nie do stylowania tylko wykorzystywania w JS, np. page-header__heading-js

jesli w LiveServer nie dziala od razu to script DEFER src="/main.js"

zmienna.innerHTML - wchodzimy do wnetrza zmiennej, mozemy ja zmienic np. header.innerHTML = 'Hej"
